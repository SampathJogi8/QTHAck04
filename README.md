# 🎛️ DSP Pro Lab – Interactive Signal Processing Playground

An advanced Streamlit-based Digital Signal Processing (DSP) simulator designed to visualize core concepts like sampling, aliasing, FFT, and noise filtering in an interactive lab-style environment.

---

## 🚀 Features

### 🔊 Signal Generator
- Sine, Square, Triangle waves
- Adjustable frequency, sampling rate, duration

### 📉 Continuous vs Sampled Signal
- Compare continuous and sampled signals

### ⚠️ Nyquist Analysis
- Detect aliasing automatically

### 🔁 Aliasing Visualization
- Compare original vs aliased signals

### 📊 FFT Spectrum Analyzer
- Frequency domain visualization

### 🎧 Audio Analyzer
- Upload WAV files and analyze waveform

### 🔇 Noise Reduction
- Apply smoothing filter to noisy signals

---

## 🧠 Concepts Covered
- Nyquist Theorem  
- Aliasing  
- FFT  
- Signal Processing Basics
